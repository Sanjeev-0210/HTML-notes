Lists

HTML lists are used to display a set of ralated items in the form of lists.

  1. Unordered List

    • < ul> tag is used for display an unordered lists of items, where there's no importance given to the order of the items.
    • All the list items are displayes using bullets.
    • Type Attributes can change the style of the List.
      1. Disc
      2. Circle
      3. Square

  2. Ordered List

    • < ol> tag is used to display the list items. When a particular order with the help of numbers.
    • The type attributes can change the style of list.
      1. Numbers(1-9)
      2. Alphabets(A/a)
      3. Roman(I/i)
    Start: This attributes specifies the starting index of the list.
    - The start value will always be an upper value.

  3. Description List

    • < dl> tag is used for display a list of Description for a particular term.
    • < dt> tag is used to define the term.
    • < dd> tag is used to display the description for the term.

Syntax:-

< ul>< ol>
< li> lists < /li>
< li>
< dl>
< dt> term < /dt>
< dd> term description < /dd>
< /dl>
< /li>
< ul>< ol>
< li> Nested list < /li>
< /ul>< /ol>
< /ul>< /ol>
