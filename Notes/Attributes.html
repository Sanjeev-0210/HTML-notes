Attributes

Attributes are properties that gives extra/additional information about a tag and also has the ability to change the behaviour of a tag. (Information about a Tag/Changes behavior)

Syntax:

< tag attribute="value">
  • Attributes:-
    1. align
    2. bgcolor(background color)
    3. src(source-url/local image)
    4. alt(alternate text, for image is missing (or) something error comes under the image)
    5. height
    6. width
E.g:
< h1 align="center">Heading of the page< /h1>
< p align="right">content< /p>
< body bgcolor="grey">< /body>
< img src="" alt="" height="" width="">

    Images:-

  • < img> tag is used to display images of the webpage.
  • It is a single tag(self closing tags).

    • Attributes of Image tag:-

    1. Src(Source)
      - This attributes specifies the path or url of the images.
    2. Alt(Alternate Text)
      - This attributes displays a text related to the image whenever the image fails to display.
    3. Height & Width
      - This attributes are used to resize an image.

    Audio & Video tag:-

  • This tags are used to display media files on the webpage.

    • Attributes:-

    1. src
    2. controls
    3. muted
    4. autoplay
    5. loop
    6. poster
    7. height
    8. width

    Note: Audio tag doesn't support poster,height and width attributes.
