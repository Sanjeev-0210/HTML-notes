Tags:-

  1. Paired tags
    These tags will be having opening tags as well as closing tags
    E.g: < h1>< /h1>,< p>< /p>,etc.
  2. Unpaired tags
    These tags will be having only opening tag and there will be no closing tag
    E.g: < br>,< img>,< hr>,etc.

Block level Elements:

  • A block-level element always starts on a new line.
  • A block-level element always takes up the full width available.
E.g:< h1>< /h1>,< h2>< /h2>,< h3>< /h3>,< h4>< /h4>,< h5>< /h5>,< h6>< /h6>,< p>< /p>

Inline Elements:

  • An inline element does not start on a new line.
  • An inline element only takes up as much width as necessary.
E.g:
  • b(Bold) or strong(Bold) tag
  • i(Italic) or em(Italic) tag
  • u(Underline) tag
  • sub(Below) tag
  • sup(power) tag
  • hr(Horizontal line) tag
  • mark(Highlight) tag
  • big(font big) tag
  • small (font small) tag
  • del(Delete) or strike(Delete) tag
  • Marquee(Moving) tag:
    - Direction(left,right,Up,Down),behavior(Scroll,Alternate,Slide),Scroll Amount(speed-10 to 100)
  • hr tag:
    - It is used to put Horizontal line in a webpage.
  • Anchor< a> tag:
    - The < a> tag defines a hyperlink, which is used to link from one page to another.
