HTML(Hyper text markup language)
Hyper- It is a text where if we click on the text it will navigate us to other webpages.
Markup- the tags we are use in html every tags are pre-defined tags.

HTML:-
It is used to create a structure of a webpage.The reason for creating a structure of webpage is to hold the information or like paragraph,heading,image etc.

Structure of HTML:-

< !DOCTYPE html>
< html lang="en">
< head>
< meta charset="UTF-8">
< meta name="viewport" content="width=device-width, initial-scale=1.0">
< title>Document< /title>
< /head>
< body>

< /body>
< /html>

Head:- We can write the tags that which we define(wanted to write) what type of data like meta tag, title tag, style tags.
Body:- It is a tag where we will be storing all the information which should be visible on the UI(User Interface).
