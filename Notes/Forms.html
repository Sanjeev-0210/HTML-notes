Forms

  • HTML forms are used to collect from user input.
  • We use < form> tag to define a form on the webpage.
    • Attributes:-
    Action: It specify the action to be perform once a form is to be submitted.

Label tag:

  • It displays a caption specify the type of values to be entered by the user.
    Attribute:-
    for- It specify the purpose of the label tag.

Input tag:

  • It creates an input field to collect the user inputs.
    Attribute:-
    1. Type: It specifies the type of value that user should enter.
      Values:- text,email,password,tel,file,checkbox,radio,date,date-timelocal,number.
    2. Placeholder: It displays a text within the < input> tag which displays before the user enters the value.
    3. Name: This attributes specifies whare the value will be stored in the database.
    4. Required: It makes an input field mandatory.
    5. Min/Max: It specifies the min and max at the range of value which will be accepted for number type input.
    6. Min length/Max length: It specifeis the length of character must be entered by the user.

Text area:

  • It is used to accept multi-line input from the user.
    Attribute:-
    1. Name: This attributes specifies whare the value will be stored in the database.
    2. Columns & Rows: Used to set the height and width of the text area.

Label tag:

  • It displays a list of option to be selected for the users.
  • Syntax:-
    < select>
    < option>we can write 'n' number of option tag inside select tag.< /option>
    < /select>

Button tag:

  • It displays a button which can be used to submit or reset the form based on the type of the button.
