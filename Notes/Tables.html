Tables

  • Tables are used to display data in the form of rows and columns.
  • < Table> tag is used to define the table on the webpages.
  • The rows for the table are created using < tr> tag.
  • < th> tag displays the heading of the table and < td> tag displays the data for the headings.

Syntax:

< table>
< tr>
< th> heading < /th>
< /tr>
< tr>
< td> data < /td>
< /tr>
< /table>

Rowspan & Colspan:

- These Attributes that allows a single cell of a table to occupay more than one row or the column.

E.g:-
< tr>
< th rowspan="3"> < /th>
< th colspan="2"> < /th>
< th rowspan="3"> < /th>
< /tr>
